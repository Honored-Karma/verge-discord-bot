## 📋 Команды по Ролям

### Администратор (ADMIN_ROLE_IDS)
**Полный доступ:**
- ✅ `/add-ap` - добавить AP
- ✅ `/set-ap` - установить AP
- ✅ `/add-sp` - добавить SP
- ✅ `/set-sp` - установить SP
- ✅ `/add-currency` - добавить валюту
- ✅ `/deduct-currency` - вычесть валюту
- ✅ `/set-currency` - установить валюту
- ✅ `/give-item` - выдать предмет
- ✅ `/give-style` - выдать стиль
- ✅ `/remove-player-style` - удалить стиль
- ✅ Все остальные админ команды

### Анкетологи (GIVE_STYLE_ROLE_IDS)
**Только управление стилями:**
- ✅ `/give-style` - выдать боевой стиль
- ✅ `/remove-player-style` - удалить стиль
- ❌ Другие команды недоступны

### Игровые Мастера (GM_ROLE_IDS)
**Управление ресурсами:**
- ✅ `/add-ap` - добавить AP
- ✅ `/set-ap` - установить AP
- ✅ `/add-sp` - добавить SP
- ✅ `/set-sp` - установить SP
- ✅ `/add-currency` - добавить валюту
- ✅ `/deduct-currency` - вычесть валюту
- ✅ `/set-currency` - установить валюту
- ✅ `/give-item` - выдать предмет
- ❌ Стили недоступны

## 🔐 Как Добавить на Хостинг

### Replit (рекомендуется)
1. Откройте проект
2. Нажмите 🔒 **Secrets** (замок слева)
3. **Add new secret** для каждой переменной:
   - Key: `ADMIN_ROLE_IDS` → Value: `1438987035862892544`
   - Key: `GIVE_STYLE_ROLE_IDS` → Value: `1431759459528610013,1438987088975626251`
   - Key: `GM_ROLE_IDS` → Value: `1431759694179074162,1438987072026185808`
4. Нажмите **Run**

### Heroku
1. Settings → **Config Vars**
2. Add для каждой переменной
3. Дайте время на перезагрузку

### Railway / Render
1. Environment variables раздел
2. Add new
3. Deploy

### Локально (.env файл)
```env
DISCORD_TOKEN=your_token
MONGODB_URI=your_connection_string

GIVE_STYLE_ROLE_IDS=1431759459528610013,1438987088975626251
```

## 🔍 Как Получить ID Роли

1. **Включите Developer Mode** в Discord
   - Settings → App Settings → Advanced → Developer Mode

2. **Скопируйте ID роли**
   - Правый клик на роль в сервере
   - Copy Role ID

3. **Вставьте в переменную окружения**

## ⚠️ Важные Напоминания

❗ **ID ролей чувствительны к опечаткам!**
Копируйте точно, без пробелов.

❗ **Не используйте имена ролей**, только ID!
(исключение: встроенная проверка "Game Master")

❗ **Разделяйте ID ТОЛЬКО запятыми БЕЗ ПРОБЕЛОВ:**

```
❌ НЕПРАВИЛЬНО: 1431759459528610013, 1438987088975626251
```
