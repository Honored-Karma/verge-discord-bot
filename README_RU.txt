ТРЕБОВАНИЯ:
═══════════════════════════════════════════════════════════════

Тебе нужно:

1. Node.js (LTS версия)
   → Скачай с https://nodejs.org
   → См. INSTALL_NODEJS.txt для подробной инструкции

2. MongoDB URI
   → Получи от MongoDB Atlas (у тебя уже есть!)
   → Должен быть в .env файле

3. Discord Token
   → Получи с https://discord.com/developers/applications
   → Должен быть в .env файле


УСТАНОВКА:
═══════════════════════════════════════════════════════════════

ШАГ 1: Установи Node.js
─────────────────────────
Если еще не установил:
  → Читай INSTALL_NODEJS.txt
  → Скачай и установи Node.js LTS


ШАГ 2: Проверь .env файл
─────────────────────────
Открой файл .env в этой папке
Убедись что есть:

    DISCORD_TOKEN=твой_токен
    CLIENT_ID=твой_id
    MONGODB_URI=mongodb+srv://...

Если MONGODB_URI некорректный:
  → Скопируй правильный из MongoDB Atlas
  → Замени <db_password> на реальный пароль


ШАГ 3: Установи зависимости
────────────────────────────
Открой PowerShell в этой папке и выполни:

    npm install

Это займет 2-3 минуты. Подожди!


ШАГ 4: Развернуть команды (первый раз)
───────────────────────────────────────
    npm run deploy

Или двойной клик на deploy.bat


ШАГ 5: Запустить бота
──────────────────────
    npm start

Или двойной клик на run.bat

Ты должен увидеть:
    ✅ Connected to MongoDB
    ✅ Database initialized successfully
    🤖 Bot is ready!


БЫСТРЫЙ СТАРТ (если всё готово):
═══════════════════════════════════════════════════════════════

Просто выполни эти команды в PowerShell:

    npm install
    npm run deploy
    npm start


ИЛИ используй батники:
  1. Двойной клик на run.bat - запустит бота
  2. Двойной клик на deploy.bat - развернуть команды

ПРОБЛЕМЫ:
═══════════════════════════════════════════════════════════════

❌ "npm not found"
  → Node.js не установлен
  → Читай INSTALL_NODEJS.txt
  → После установки перезагрузи PowerShell!

❌ "MONGODB_URI not found"
  → Добавь MONGODB_URI в .env
  → Убедись что пароль правильный

❌ "connect ECONNREFUSED"
  → Проверь интернет
  → Проверь MONGODB_URI в .env
  → Проверь что MongoDB Atlas доступна

❌ "not authorized"
  → Неверный пароль в MONGODB_URI
  → Скопируй правильный с MongoDB Atlas

❌ Батники (.bat) не запускаются
  → Может быть блокировка Windows
  → Используй PowerShell вместо батников
  → Команда: npm install
