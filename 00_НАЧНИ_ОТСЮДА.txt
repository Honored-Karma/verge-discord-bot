🚀 ГОТОВИМСЯ К ЗАПУСКУ (5 МИНУТ)

ШАГ 1: Проверь .env файл
─────────────────────────────────────────────
Открой файл .env и убедись что есть:

    DISCORD_TOKEN=MTQ0MTg2NDcyMjQzMjY1OTcwOA.G7tIOJ...
    CLIENT_ID=1441864722432659708
    MONGODB_URI=mongodb+srv://zerokarma08_db_user:ПАРОЛЬ@...

⚠️  ВАЖНО: Замени ПАРОЛЬ на реальный пароль от MongoDB!


ШАГ 2: Установи зависимости
─────────────────────────────────────────────
Открой терминал в папке проекта и выполни:

    npm install

Это установит mongodb драйвер и все остальные пакеты.


ШАГ 3: Развей команды на сервер
─────────────────────────────────────────────
Первый раз нужно развернуть команды:

    npm run deploy

После этого все слэш-команды будут доступны в Discord.


ШАГ 4: Запусти бота
─────────────────────────────────────────────
    npm start

Если все правильно, ты увидишь:

    ✅ Connected to MongoDB
    ✅ Database initialized successfully
    🤖 Bot is ready! Logged in as YourBot#0000

🗂️ СТРУКТУРА MONGODB

Твоя MongoDB будет иметь эти коллекции (создаются автоматически):

discord_bot (database)
  ├─ players
  │   ├─ id (unique)
  │   ├─ username
  │   ├─ character_name
  │   ├─ ap, krw, yen
  │   ├─ ap_multiplier, sp_multiplier
  │   └─ И другие поля...
  │
  ├─ styles
  │   ├─ id (unique)
  │   ├─ name (unique)
  │   ├─ created_by
  │   └─ created_at
  │
  ├─ player_sp
  │   ├─ player_id + style_id (composite unique)
  │   └─ sp
  │
  ├─ inventory
  │   ├─ player_id
  │   ├─ item_name
  │   └─ qty
  │
  └─ admin_actions
      ├─ admin_id
      ├─ action
      ├─ details
      └─ timestamp

❓ ЕСЛИ ЕСТЬ ПРОБЛЕМЫ

Проблема: "MONGODB_URI not found"
Решение: Добавь MONGODB_URI в .env

Проблема: "connect ECONNREFUSED"
Решение: Проверь интернет и URL в .env

Проблема: "not authorized"
Решение: Неверный пароль в MONGODB_URI

Проблема: "timeout waiting for replication"
Решение: Медленный интернет, попробуй позже
