# 📢 Команда `/announce` - Система массовых рассылок

## 📋 Что было создано

Для вашего Discord бота была создана полнофункциональная система массовых рассылок с красивыми embed-сообщениями в личные сообщения участников сервера.

### Созданные файлы:

1. **`commands/announce.js`** - Основная команда
2. **`ANNOUNCE_COMMAND_GUIDE.md`** - Полная документация
3. **`ANNOUNCE_QUICKSTART.md`** - Быстрый старт
4. **`ANNOUNCE_README.md`** - Этот файл

### Обновлённые файлы:

1. **`commands/help.js`** - Добавлена информация о команде `/announce`

---

## ⚡ Основные возможности

### ✅ Функционал

- **Массовая рассылка** всем участникам сервера
- **Выборочная отправка** одному участнику
- **Групповая отправка** нескольким участникам по ID
- **Красивые Embed** с настраиваемым дизайном
- **Превью** сообщения перед отправкой
- **Подтверждение** с кнопками (60 секунд)
- **Детальный отчёт** о результатах рассылки
- **Защита от rate limit** (задержка 500мс)
- **Автофильтрация ботов**

### 🎨 Настройки дизайна

- Заголовок (title)
- Цвет боковой полосы (color в HEX)
- Большое изображение (image)
- Миниатюра (thumbnail)
- Футер (footer)
- Основной текст (message)

### 🛡️ Безопасность

- Доступ только для администраторов
- Двойное подтверждение перед отправкой
- Таймаут подтверждения (60 секунд)
- Защита от спама (задержка между сообщениями)
- Валидация параметров

## 📖 Примеры использования

### Пример 1: Простое объявление всем
```
/announce 
  message: Всем привет! Сегодня в 20:00 важное событие!
  target: Всем участникам сервера
```

**Результат:** Все участники сервера (кроме ботов) получат embed в ЛС с текстом объявления.

---

### Пример 2: Объявление с кастомизацией
```
/announce 
  message: Новый сезон начинается через неделю! Приготовьтесь к эпическим битвам!
  target: Всем участникам сервера
  title: 🌟 Сезон 2 уже скоро!
  color: #00ff00
  image: https://example.com/season2-banner.png
  footer: Команда Verge RPG
```

**Результат:** Красивое embed с зелёной полосой, баннером и кастомным заголовком.

---

### Пример 3: Персональное сообщение
```
/announce 
  message: Поздравляем! Вы стали победителем турнира и получаете специальную награду!
  target: Выбранному участнику
  user: @JohnDoe
  title: 🏆 Поздравляем с победой!
  color: #ffd700
```

**Результат:** Только @JohnDoe получит личное поздравительное сообщение.

---

### Пример 4: VIP приглашение нескольким участникам
```
/announce 
  message: Вы приглашены на закрытое VIP мероприятие завтра в 19:00!
  target: Выбранным участникам
  users: 123456789012345678, 987654321098765432, 111222333444555666
  title: 🔒 VIP Приглашение
  color: #9900ff
  thumbnail: https://example.com/vip-icon.png
```

**Результат:** Только указанные пользователи получат приглашение.

## 📊 Параметры команды

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `message` | String | ✅ Да | Текст объявления (до 4096 символов) |
| `target` | Choice | ✅ Да | Кому отправить: all/single/multiple |
| `user` | User | ❌ Нет | Пользователь (только для target=single) |
| `users` | String | ❌ Нет | ID через запятую (только для target=multiple) |
| `title` | String | ❌ Нет | Заголовок (по умолчанию: 📢 Объявление) |
| `color` | String | ❌ Нет | HEX цвет (по умолчанию: #b209d4) |
| `image` | String | ❌ Нет | URL большого изображения |
| `thumbnail` | String | ❌ Нет | URL миниатюры |
| `footer` | String | ❌ Нет | Текст footer (по умолчанию: название сервера) |

## 🔧 Технические детали

### Архитектура
- **Framework:** Discord.js v14+
- **Тип команды:** Slash Command
- **Права:** Administrator только
- **Rate Limiting:** 500мс между сообщениями

### Обработка ошибок
```javascript
// Автоматически обрабатывает:
- Закрытые ЛС пользователей
- Несуществующих пользователей
- Rate limit Discord API
- Неверные форматы параметров
- Таймауты подтверждения
```

### Производительность
- **100 пользователей:** ~50 секунд
- **500 пользователей:** ~4 минуты
- **1000 пользователей:** ~8 минут

## 📈 Процесс отправки

```
1. Администратор вводит команду
        ↓
2. Бот валидирует параметры
        ↓
3. Создаётся список получателей
        ↓
4. Показывается превью с кнопками
        ↓
5. Администратор подтверждает/отменяет
        ↓
6. Начинается рассылка (с задержкой)
        ↓
7. Подсчитываются успешные/неуспешные отправки
        ↓
8. Показывается детальный отчёт
```

## ⚠️ Важные замечания

### Ограничения Discord
- Пользователи с закрытыми ЛС не получат сообщение
- Embed максимум 4096 символов в description
- Rate limit: ~50 сообщений в 10 секунд (мы используем 2/секунду)

### Рекомендации
1. **Тестируйте на себе** - отправьте сообщение себе перед массовой рассылкой
2. **Краткость** - делайте сообщения короткими и понятными
3. **Расписание** - не злоупотребляйте рассылками
4. **Проверка** - проверяйте правильность ID перед отправкой

### Что НЕ делать
- ❌ Спамить рассылками
- ❌ Отправлять оскорбительный контент
- ❌ Рассылать без согласия участников
- ❌ Использовать для рекламы

## 🐛 Решение проблем

### Команда не появляется в Discord
```bash
# Решение 1: Передеплойте команды
npm run deploy

# Решение 2: Перезапустите бота
npm start
```

### Сообщения не доставляются
- Проверьте настройки приватности пользователей
- Убедитесь, что бот имеет права отправлять ЛС
- Проверьте, не заблокирован ли бот пользователем

### Ошибка "No permissions"
- Убедитесь, что вы администратор сервера
- Проверьте права бота на сервере
