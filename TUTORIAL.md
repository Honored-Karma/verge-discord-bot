## Что такое Verge?

**Verge** - это полнофункциональный Discord RPG бот, который позволяет вам:

✨ Создавать персонажей  
💪 Тренироваться в боевых стилях  
💰 Зарабатывать и обменивать валюту  
🏆 Соревноваться в рейтингах  
👥 Играть с друзьями  

### Основные цифры
- **24+ команд** для полного контроля
- **8+ боевых стилей** для изучения
- **Двойная валюта** (KRW и YEN) для экономики
- **MongoDB** для надёжного хранения данных
- **100% на русском** языке

## Начало работы

### Шаг 1: Регистрация

Используйте команду:
```
/register
```

**Параметры:**
- `character_name` - Придумайте имя для своего персонажа
- `avatar_url` (опционально) - URL на изображение для аватара

**Пример:**
```
/register character_name:"Воин света" avatar_url:"https://example.com/avatar.jpg"
```

**Что происходит после регистрации:**
✅ Создаётся ваш персонаж  
✅ Вы начинаете с 0 AP (Attack Power)  
✅ Вы получаете доступ ко всем стилям  
✅ Ваши данные сохраняются в базе данных  

### Шаг 2: Выберите стиль боя

Смотрите доступные стили:
```
/styles-list
```

Начните тренировку в интересующем вас стиле:
```
/train Muay Thai
```

### Шаг 3: Смотрите свой прогресс

Откройте свой профиль:
```
/profile
```

**Что вы увидите:**
- 📊 Ваш AP (общая сила)
- 🥋 SP в каждом стиле
- 💰 Ваш баланс валюты
- 👥 Статистика по командам

## Система персонажа

### AP (Attack Power) - Сила атаки

**Что это:**  
Общая характеристика вашего персонажа, которая растёт при тренировке.

**Как получить:**
- 📈 `+10 AP` за каждую тренировку (`/train`)
- 📈 `+20 AP` за социальный RP (`/social-rp`)

**Для чего нужен:**
- 🏆 Влияет на ваш рейтинг в лидерборде
- 💪 Показывает общий уровень развития
- 🎯 Требуется для некоторых стилей (50+ AP для продвинутых)

**Множители:**  
Администраторы могут установить множитель AP (по умолчанию 100%):
- 150% = получаете в 1.5 раза больше AP
- 50% = получаете в 2 раза меньше AP

## Система боевых стилей

### Что такое SP (Style Points)?

**SP** - это очки мастерства в каждом конкретном стиле боя.

**Как получить SP:**
1. Используйте `/train [стиль]` чтобы тренироваться в стиле
   - Получаете `+30 SP` в выбранный стиль
   - Кулдаун: 5 часов

2. Используйте `/social-rp [действие]` для социального RP
   - Получаете `+15 SP` в случайный стиль
   - Кулдаун: 12 часов

### Ранги мастерства

По мере накопления SP в стиле, вы получаете ранги:

| Ранг | SP | Описание |
|------|-----|---------|
| 🌱 Новичок (Novice) | 0-349 | Вы только начинаете |
| 👤 Владелец (Owner) | 350-999 | Хороший уровень владения |
| 🎯 Эксперт (Expert) | 1000-2499 | Глубокие знания стиля |
| 👑 Мастер (Master) | 2500+ | Абсолютное мастерство |

### Социальный RP

Помимо традиционной тренировки, вы можете получать очки через социальный RP:

```
/social-rp медитирую в святилище
```

**Правила:**
- Награда: `+20 AP` и `+15 SP` в случайный стиль
- Кулдаун: 12 часов
- Должно быть описание действия (минимум несколько слов)

**Примеры:**
- "практикую формы вместе с сенсеем"
- "участвую в турнире"
- "учу новичков боевым искусствам"

## Система экономики

### Валюты

Verge использует **двойную валютную систему**:

#### 🇰🇷 KRW (Корейская вона)
Основная валюта для торговли и транзакций.

#### 🇯🇵 YEN (Японская иена)
Премиум валюта для редких товаров и услуг.

### Как получить деньги

1. **Награды за команды** (будущий функционал)
2. **От админов** через `/add-currency` или `/set-currency`
3. **Обмен с другими игроками** через `/pay`

### Команды с деньгами

#### `/pay` - Отправить деньги

Отправьте деньги другому игроку:
```
/pay user:@John currency:yen amount:100
```

**Параметры:**
- `user` - Получатель платежа
- `currency` - Тип валюты (krw или yen)
- `amount` - Сумма

**Условия:**
- Нужно иметь достаточно денег
- Без комиссии
- Мгновенный перевод

#### `/exchange` - Обменять валюту

Обменяйте одну валюту на другую:
```
/exchange from:krw to:yen amount:1000
```

**Параметры:**
- `from` - Из какой валюты (krw или yen)
- `to` - В какую валюту
- `amount` - Сумма к обмену

**Курсы:**
- Курс динамический и меняется
- Нет фиксированного значения
- Администраторы могут управлять курсом

## Профиль и инвентарь

### Просмотр профиля

Откройте свой профиль:
```
/profile
```

Или профиль другого игрока:
```
/profile user:@John
```

**Что показывает профиль:**
1. **Основная информация**
   - Имя персонажа
   - Аватар
   - Общий AP

2. **Статистика по стилям**
   - SP в каждом стиле
   - Ранг в каждом стиле
   - Прогресс к следующему рангу

3. **Финансовая информация**
   - Баланс KRW
   - Баланс YEN
   - Общее богатство

4. **Статистика**
   - Дата регистрации
   - Количество тренировок
   - Уровень активности

### Смена аватара ✨ НОВОЕ!

Установите новый аватар для вашего персонажа:
```
/set-avatar url:https://example.com/avatar.jpg
```

**Поддерживаемые форматы:**
- JPG, PNG, GIF, WebP
- Discord CDN ссылки
- Большинство хостингов изображений

**Примеры валидных ссылок:**
- https://example.com/image.jpg
- https://cdn.discordapp.com/avatars/...jpg
- https://imgur.com/image.png

### Инвентарь

Просмотрите ваш инвентарь:
```
/inventory
```

Или инвентарь другого игрока:
```
/inventory user:@John
```

**Что в инвентаре:**
- 🎁 Предметы
- 📦 Количество каждого
- 🔧 Возможность использования

### Использование предметов

Используйте предмет из инвентаря:
```
/use item:AP Tome (50)
```

**Примеры предметов:**
- `AP Tome (50)` - Даёт `+50 AP`
- `AP Tome (100)` - Даёт `+100 AP`
- `SP Scroll: Muay Thai` - Даёт `+30 SP` в Muay Thai

## Продвинутая игра

### Лидерборды

Смотрите рейтинги лучших игроков:
```
/leaderboard type:ap
```

**Доступные рейтинги:**
- `ap` - По общей силе (Attack Power)
- `sp` - По мастерству (Style Points)
- `krw` - По корейской воне
- `yen` - По японской иене

**Что это даёт:**
🏆 Соревновательный элемент  
👑 Репутация и слава  
🎯 Мотивация для прогресса  

### Множители

Администраторы могут установить **множители** для всех игроков:

#### AP Multiplier
```
/set-ap-multiplier multiplier:150
```
- 150 = получаете в 1.5 раза больше AP
- 200 = получаете в 2 раза больше AP
- 50 = получаете в 2 раза меньше AP

#### SP Multiplier
```
/set-sp-multiplier multiplier:150
```
- Работает так же, но для SP

### Стратегия прогресса

#### Способ 1: Фокус на одном стиле
1. Выберите один стиль боя
2. Тренируйтесь в нём несколько раз подряд
3. Достигните ранга Мастер
4. Затем переходите на другой стиль

**Преимущества:**
✅ Быстрее достичь высокого ранга  
✅ Глубокое мастерство  

**Недостатки:**
❌ Развитие идёт медленнее в других стилях  

#### Способ 2: Разностороннее развитие
1. Тренируйтесь поочерёдно в разных стилях
2. Развивайте все стили равномерно
3. Получайте общий уровень AP быстрее

**Преимущества:**
✅ Универсальность  
✅ Быстрый общий уровень  

**Недостатки:**
❌ Дольше достигать высоких рангов  

#### Способ 3: Социальный фокус
1. Используйте `/social-rp` для случайных стилей
2. Комбинируйте с целевыми тренировками
3. Получайте AP и SP одновременно

**Преимущества:**
✅ Разнообразие  
✅ Больше контроля

## FAQ

### ❓ Как часто я могу тренироваться?

**Ответ:**  
Каждая команда имеет кулдаун:
- `/train` - 5 часов между использованием
- `/social-rp` - 12 часов между использованием
- Другие команды - ~1 секунда между ними

### ❓ Что такое "ephemeral" сообщение?

**Ответ:**  
Это приватное сообщение, которое видите только вы. Оно автоматически удаляется через 15 минут.

### ❓ Как получить много денег?

**Ответ:**  
Администраторы могут выдать вам деньги через `/add-currency` или `/set-currency`. В будущих версиях будут миссии и квесты.

### ❓ Можно ли удалить аватар?

**Ответ:**  
Используйте любой другой URL аватара или URL на стандартный аватар Discord.

### ❓ Что произойдёт, если я неправильно напишу стиль боя?

**Ответ:**  
Бот выдаст ошибку. Используйте `/styles-list` чтобы видеть точные названия.

### ❓ Сколько предметов может быть в инвентаре?

**Ответ:**  
Не ограничено! Вы можете собирать сколько угодно.

### ❓ Как стать администратором?

**Ответ:**  
Владелец сервера может дать вам роль или ID администратора. Спросите владельца!

### ❓ Что, если я потеряю доступ к аккаунту?

**Ответ:**  
Ваши данные хранятся в базе данных и привязаны к Discord ID. Если вы вернёте доступ - данные восстановятся.

### ❓ Можно ли торговать с другими игроками?

**Ответ:**  
Пока нет встроенной торговли, но вы можете обмениваться через `/pay` для денег.

### ❓ Как часто обновляется бот?

**Ответ:**  
Обновления выходят примерно раз в месяц с новыми командами и улучшениями.
