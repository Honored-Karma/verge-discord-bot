## 🛠️ Возможные проблемы и решения

### Проблема 1: Команда не появляется

**Причина:** Команды не задеплоены или бот не перезапущен

**Решение:**
```bash
# 1. Деплой команд
npm run deploy

# 2. Перезапуск бота
npm start
```

---

### Проблема 2: Ошибка "No permissions"

**Причина:** Вы не администратор сервера

**Решение:**
- Убедитесь, что у вас есть права администратора на сервере
- Проверьте настройки ролей в Discord

---

### Проблема 3: Бот не отправляет ЛС

**Причина:** У получателя закрыты личные сообщения

**Решение:**
- Попросите получателя включить ЛС в настройках приватности сервера
- Настройки → Приватность и безопасность → Прямые сообщения от участников сервера ✅

---

### Проблема 4: Ошибка при деплое

**Причина:** Неверные переменные окружения

**Решение:**
Проверьте файл `.env`:
```
DISCORD_TOKEN=ваш_токен_бота
CLIENT_ID=id_вашего_бота
GUILD_ID=id_сервера (опционально)
```
