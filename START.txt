⚡ БЫСТРЫЙ СТАРТ (3 шага)

1. Убедись что в .env есть:
   DISCORD_TOKEN=твой_токен
   CLIENT_ID=твой_id  
   MONGODB_URI=mongodb+srv://...

2. Установи зависимости:
   npm install

3. Запусти бота:
   npm start
