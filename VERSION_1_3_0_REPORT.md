## 📦 Что изменилось в боте?

### Новая команда
```javascript
// commands/help.js
export const data = new SlashCommandBuilder()
    .setName('help')
    .setDescription('Полный гайд по всем командам бота')
    .addStringOption(option =>
        option.setName('category')
            .setDescription('Выбрать категорию команд')
            .setRequired(false)
            // 5 категорий с выбором
    );
```

### Функциональность
- ✅ Выпадающее меню с 6 опциями
- ✅ 5 красивых embed-ов для каждой категории
- ✅ Описание каждой команды
- ✅ Примеры использования
