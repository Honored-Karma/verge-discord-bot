# 📢 Руководство по команде `/announce`

## Описание
Команда `/announce` позволяет администраторам отправлять красивые объявления в личные сообщения участникам сервера. Сообщения отправляются в формате Embed с настраиваемым дизайном.

## Права доступа
⚠️ **Только администраторы сервера** могут использовать эту команду!

## Параметры команды

### Обязательные параметры:

#### `message` (обязательный)
Основной текст объявления, который будет отображаться в embed.

**Пример:** `Привет! Сегодня в 20:00 состоится турнир!`

#### `target` (обязательный)
Выбор целевой аудитории для рассылки:
- **Всем участникам сервера** - отправить всем участникам (кромеботов)
- **Выбранному участнику** - отправить одному конкретному участнику
- **Выбранным участникам** - отправить нескольким участникам по ID

### Дополнительные параметры:

#### `user` (опциональный)
Упоминание пользователя для отправки.
- **Используется когда:** `target = "Выбранному участнику"`
- **Формат:** @username или выбор из списка

#### `users` (опциональный)
ID пользователей через запятую для отправки нескольким участникам.
- **Используется когда:** `target = "Выбранным участникам"`
- **Формат:** `123456789012345678, 987654321098765432, 111222333444555666`

#### `title` (опциональный)
Заголовок объявления в embed.
- **По умолчанию:** `📢 Объявление`
- **Пример:** `🎉 Важное событие!`

#### `color` (опциональный)
Цвет боковой полосы embed в HEX формате.
- **По умолчанию:** `#b209d4` (фиолетовый)
- **Формат:** `#RRGGBB` или `#RGB`
- **Примеры:** 
  - `#ff0000` - красный
  - `#00ff00` - зелёный
  - `#0000ff` - синий
  - `#ffaa00` - оранжевый

#### `image` (опциональный)
URL большого изображения, которое будет отображаться в embed.
- **Формат:** Прямая ссылка на изображение
- **Пример:** `https://example.com/banner.png`

#### `thumbnail` (опциональный)
URL миниатюры (маленькое изображение в правом верхнем углу).
- **Формат:** Прямая ссылка на изображение
- **Пример:** `https://example.com/icon.png`

#### `footer` (опциональный)
Текст в нижней части embed.
- **По умолчанию:** Название сервера
- **Пример:** `Администрация Verge RPG`

## Примеры использования

### Пример 1: Отправка объявления всем участникам
```
/announce 
  message: Внимание! Сегодня в 20:00 состоится турнир!
  target: Всем участникам сервера
  title: 🎮 Турнир
  color: #ff6600
```

### Пример 2: Отправка одному участнику
```
/announce 
  message: Привет! Ты был выбран для участия в ивенте!
  target: Выбранному участнику
  user: @JohnDoe
  title: ⭐ Персональное приглашение
  color: #ffd700
```

### Пример 3: Отправка нескольким участникам
```
/announce 
  message: Вы приглашены на закрытый ивент!
  target: Выбранным участникам
  users: 123456789012345678, 987654321098765432
  title: 🔒 VIP Приглашение
  color: #9900ff
```

### Пример 4: Объявление с изображением
```
/announce 
  message: Новый сезон начинается через неделю! Приготовьтесь!
  target: Всем участникам сервера
  title: 🌟 Новый сезон!
  color: #00ffff
  image: https://example.com/season_banner.png
  thumbnail: https://example.com/logo.png
  footer: Команда разработки Verge RPG
```

## Процесс отправки

1. **Введите команду** с нужными параметрами
2. **Превью** - бот покажет превью вашего объявления
3. **Подтверждение** - нажмите кнопку "✅ Отправить" или "❌ Отмена"
4. **Отправка** - бот начнёт рассылку (появится индикатор прогресса)
5. **Результат** - получите отчёт о результатах рассылки

## Особенности

### ✅ Преимущества:
- Красивые Embed сообщения
- Превью перед отправкой
- Подтверждение отправки
- Защита от случайной отправки (60 секунд на подтверждение)
- Подробный отчёт о результатах
- Защита от rate limit (задержка 500мс между сообщениями)
- Автоматическая фильтрацияботов

### ⚠️ Важно знать:
- Команда доступна только администраторам
- Боты автоматически исключаются из рассылки
- Если у пользователя закрыты личные сообщения, он не получит объявление
- При больших рассылках процесс может занять время
- Отчёт покажет, кому не удалось доставить сообщение

### 🛡️ Защита от спама:
- Задержка 500мс между отправками
- Только администраторы могут использовать команду
- Требуется подтверждение перед отправкой
- 60 секунд на принятие решения

## Частые ошибки и решения

### ❌ "У вас нет прав для использования этой команды"
**Решение:** Эта команда доступна только администраторам сервера.

### ❌ "Неверный формат цвета"
**Решение:** Используйте HEX формат: `#b209d4` или `#f00`

### ❌ "Вы должны указать пользователя для отправки"
**Решение:** При выборе режима "Выбранному участнику" укажите параметр `user`

### ❌ "Не удалось найти ни одного пользователя"
**Решение:** Проверьте правильность ID пользователей в параметре `users`

### ⚠️ Некоторые пользователи не получили сообщение
**Причина:** У них закрыты личные сообщения от участников сервера
**Решение:** Попросите их включить ЛС в настройках конфиденциальности сервера

## Советы по использованию

1. **Тестируйте на себе** - сначала отправьте объявление себе, чтобы проверить как оно выглядит
2. **Используйте изображения** - визуальный контент привлекает больше внимания
3. **Краткость** - старайтесь делать сообщения короткими и понятными
4. **Цветовая схема** - используйте цвета, которые соответствуют типу сообщения:
   - 🔴 Красный (#ff0000) - срочно/важно
   - 🟢 Зелёный (#00ff00) - позитивные новости
   - 🔵 Синий (#0000ff) - информация
   - 🟡 Жёлтый (#ffff00) - предупреждение
   - 🟣 Фиолетовый (#b209d4) - стандартные объявления

## Технические детали

- **Rate Limit Protection:** 500мс между сообщениями
- **Timeout подтверждения:** 60 секунд
- **Максимум получателей:** Нет ограничений (но учитывайте время)
- **Поддерживаемые форматы изображений:** PNG, JPG, GIF, WebP
- **Максимальная длина сообщения:** 4096 символов (ограничение Discord Embed)
