ВАРИАНТ 1: REPLIT (РЕКОМЕНДУЕТСЯ - САМЫЙ ПРОСТОЙ)
─────────────────────────────────────────────────────────────────────────────────
1. Откройте свой проект на Replit.com
2. На левой панели нажмите иконку 🔒 "Secrets"
3. Нажмите "Add new secret"
4. Добавьте каждую переменную:
   - Key: ADMIN_ROLE_IDS
     Value: 1438987035862892544
   - Key: GIVE_STYLE_ROLE_IDS
     Value: 1431759459528610013,1438987088975626251
   - Key: GM_ROLE_IDS
     Value: 1431759694179074162,1438987072026185808
5. Не забудьте добавить DISCORD_TOKEN и MONGODB_URI
6. Нажмите "Run" для перезагрузки


ВАРИАНТ 2: HEROKU
─────────────────────────────────────────────────────────────────────────────────
1. Откройте приложение на Heroku.com
2. Settings → Config Vars
3. Нажмите "Reveal Config Vars"
4. Нажмите "Add"
5. Добавьте переменные по одной
6. Дайте время на перезагрузку


ВАРИАНТ 3: RAILWAY / RENDER / ДРУГИЕ
─────────────────────────────────────────────────────────────────────────────────
1. Найдите раздел Environment Variables в вашем хостинге
2. Добавьте переменные
3. Нажмите Deploy / Save
