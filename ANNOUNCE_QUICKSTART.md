# 🚀 Быстрый старт: Команда `/announce`

## Установка

Команда уже создана! Осталось только задеплоить её:

### Шаг 1: Деплой команды
```bash
npm run deploy
# или
node deploy-commands.js
```

### Шаг 2: Перезапуск бота (если уже запущен)
```bash
# Остановите бота (Ctrl+C)
# Затем запустите снова
npm start
# или
node index.js
```

## Быстрые примеры использования

### 📢 Объявление всем участникам
```
/announce 
  message: Всем привет! Сегодня в 20:00 важный ивент!
  target: Всем участникам сервера
```

### 👤 Персональное сообщение
```
/announce 
  message: Поздравляем! Вы выиграли приз!
  target: Выбранному участнику
  user: @Username
  title: 🎉 Поздравляем!
  color: #ffd700
```

### 👥 Сообщение нескольким участникам
```
/announce 
  message: Вы приглашены на закрытую встречу
  target: Выбранным участникам
  users: 123456789012345678, 987654321098765432
  title: 🔒 VIP Приглашение
```

### 🎨 Объявление с картинкой
```
/announce 
  message: Новый сезон начинается!
  target: Всем участникам сервера
  title: 🌟 Сезон 2
  color: #00ff00
  image: https://example.com/banner.png
```

## Основные параметры

| Параметр | Обязательный | Описание |
|----------|--------------|----------|
| `message` | ✅ Да | Текст объявления |
| `target` | ✅ Да | Кому отправить (all/single/multiple) |
| `user` | ❌ Нет | Пользователь (если target=single) |
| `users` | ❌ Нет | ID через запятую (если target=multiple) |
| `title` | ❌ Нет | Заголовок (по умолчанию: 📢 Объявление) |
| `color` | ❌ Нет | HEX цвет (по умолчанию: #b209d4) |
| `image` | ❌ Нет | URL большого изображения |
| `thumbnail` | ❌ Нет | URL миниатюры |
| `footer` | ❌ Нет | Текст footer (по умолчанию: название сервера) |

## Как получить ID пользователя

1. Включите режим разработчика в Discord:
   - Настройки → Расширенные → Режим разработчика ✅
2. Нажмите ПКМ на пользователя → Копировать ID

## Цветовая палитра (HEX коды)

- 🔴 Красный (важно): `#ff0000`
- 🟢 Зелёный (успех): `#00ff00`
- 🔵 Синий (инфо): `#0000ff`
- 🟡 Жёлтый (внимание): `#ffff00`
- 🟣 Фиолетовый (стандарт): `#b209d4`
- 🟠 Оранжевый: `#ff6600`
- 🩷 Розовый: `#ff69b4`
- 🤎 Коричневый: `#8b4513`
- ⚫ Чёрный: `#000000`
- ⚪ Белый: `#ffffff`

## Важные моменты

✅ **Что работает:**
- Отправка всем участникам сервера
- Выборочная отправка одному или нескольким участникам
- Настройка дизайна embed (цвет, картинки, заголовок)
- Превью перед отправкой
- Подтверждение отправки
- Подробный отчёт о результатах

⚠️ **Что нужно знать:**
- Команда доступна только администраторам
- Если у пользователя закрыты ЛС, он не получит сообщение
- Боты автоматически исключаются из рассылки
- Между сообщениями задержка 500мс (защита от rate limit)
- У вас есть 60 секунд на подтверждение отправки

## Частые вопросы

**Q: Почему некоторые не получили сообщение?**  
A: У них закрыты личные сообщения. Попросите их включить ЛС в настройках приватности сервера.

**Q: Сколько времени займёт рассылка 100 участникам?**  
A: Примерно 50 секунд (500мс между сообщениями).

**Q: Можно ли отменить отправку?**  
A: Да! Нажмите кнопку ❌ Отмена в превью или просто дождитесь таймаута (60 сек).

**Q: Как найти ID пользователя?**  
A: Включите режим разработчика в Discord, затем ПКМ → Копировать ID.

2. Убедитесь что команда задеплоена
