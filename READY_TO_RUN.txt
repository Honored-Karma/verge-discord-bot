ВАРИАНТ 1: Батники (проще всего)
─────────────────
1. Убедись что установлен Node.js
   → Если нет: читай INSTALL_NODEJS.txt

2. Двойной клик на run.bat
   → Установит зависимости и запустит бота

3. Первый раз перед этим:
   → Двойной клик на deploy.bat

ВАРИАНТ 2: PowerShell (стандартный способ)
──────────────────
1. Открой PowerShell в этой папке

2. Установи зависимости:
   npm install

3. Развернуть команды (первый раз):
   npm run deploy

4. Запустить бота:
   npm start


⚠️ ПЕРЕД ЗАПУСКОМ:
─────────────────────────────────────────────────────────────

1. Установи Node.js (если еще не установлен)
   → Ссылка: https://nodejs.org (выбери LTS)
   → Инструкция: читай INSTALL_NODEJS.txt

2. Проверь .env файл
   → DISCORD_TOKEN должен быть корректный
   → CLIENT_ID должен быть корректный
   → MONGODB_URI должен быть корректный
     (замени <db_password> на реальный пароль)

3. Интернет должен быть
   → Для подключения к MongoDB
   → Для подключения к Discord

🎯 ЕСЛИ ЕСТЬ ПРОБЛЕМЫ:
─────────────────────────────────────────────────────────────

❌ "npm not found"
   → Node.js не установлен
   → Прочитай INSTALL_NODEJS.txt
   → Перезагрузи PowerShell после установки!

❌ "ECONNREFUSED" или ошибки MongoDB
   → Проверь MONGODB_URI в .env
   → Замени <db_password> на реальный пароль
   → Проверь интернет

❌ "not authorized"
   → Неверный пароль в MONGODB_URI
   → Скопируй правильный с MongoDB Atlas

❌ Батники не запускаются
   → Может быть блокировка Windows
   → Используй PowerShell вместо батников

❌ "This script requires admin rights"
   → setup.ps1 требует прав администратора
   → Используй батники или npm команды в PowerShell
