# 🎨 Визуальные примеры команды `/announce`

Этот документ показывает, как будут выглядеть различные варианты embed сообщений.

---

## 📢 Пример 1: Стандартное объявление

### Команда:
```
/announce 
  message: Всем привет! Сегодня в 20:00 состоится важное событие на сервере. Не пропустите!
  target: Всем участникам сервера
```

### Как выглядит в Discord:

```
┌─────────────────────────────────────────┐
│ 📢 Объявление                            │
├─────────────────────────────────────────┤
│                                          │
│ Всем привет! Сегодня в 20:00 состоится  │
│ важное событие на сервере. Не пропустите!│
│                                          │
├─────────────────────────────────────────┤
│ Название Сервера • Сегодня в 15:30      │
└─────────────────────────────────────────┘
```
**Цвет:** 🟣 Фиолетовый (#b209d4)

## 💡 Советы по дизайну

### ✅ Хорошие практики:

1. **Соответствие цвета и контента**
   - Красный для срочного
   - Зелёный для позитивного
   - Жёлтый для предупреждений

2. **Использование emoji**
   - Добавляйте emoji в заголовок
   - Не перебарщивайте (1-2 достаточно)

3. **Изображения**
   - Используйте качественные изображения
   - Оптимальный размер: 1920x1080 или 16:9
   - Миниатюры: квадратные, 512x512

4. **Длина текста**
   - Краткость - сестра таланта
   - Максимум 3-4 предложения
   - Важное - в начало

### ❌ Чего избегать:

1. Слишком яркие кислотные цвета
2. Длинные простыни текста
3. Много разных цветов в одном сообщении
4. Битые ссылки на изображения
5. Слишком частые рассылки
