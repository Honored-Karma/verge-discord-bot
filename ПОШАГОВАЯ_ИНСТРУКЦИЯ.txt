ПОШАГОВАЯ ИНСТРУКЦИЯ:

════════════════════════════════════════════════════════════════

⚠️  ВАЖНО: ПЕРВЫЙ ШАГ - УСТАНОВИ Node.js!

ШАГ 1: Установи Node.js (КРИТИЧНО!)
─────────────────────────────────────
Если у тебя еще не установлен Node.js:

1. Открой файл INSTALL_NODEJS.txt (он в этой папке)
2. Следуй пошаговой инструкции
3. Скачай Node.js с https://nodejs.org (выбери LTS)
4. Установи обычным способом
5. ⚠️ ПЕРЕЗАГРУЗИ PowerShell!

После установки проверь:
  Открой PowerShell и выполни:
  node --version

Если видишь версию (например v20.10.0) - отлично!


ШАГ 2: Проверь .env файл
───────────────────────────
1. Открой файл .env в этой папке (в любом редакторе)

2. Убедись что там есть эти строки:
   DISCORD_TOKEN=твой_токен
   CLIENT_ID=твой_id
   MONGODB_URI=mongodb+srv://...

3. Если MONGODB_URI неправильный:
   • Открой MongoDB Atlas
   • Скопируй CONNECTION STRING
   • Вставь в .env
   • Замени <password> на реальный пароль


ШАГ 3: Установи зависимости
──────────────────────────────
Выполни одно из:

ВАРИАНТ А (проще):
  Двойной клик на run.bat
  (он все сделает автоматически)

ВАРИАНТ Б (в PowerShell):
  1. Открой PowerShell в этой папке
  2. Выполни:
     npm install
  3. Подожди 2-3 минуты


ШАГ 4: Развернуть команды (ПЕРВЫЙ РАЗ)
────────────────────────────────────
Выполни одно из:

ВАРИАНТ А (проще):
  Двойной клик на deploy.bat

ВАРИАНТ Б (в PowerShell):
  npm run deploy

Команды будут доступны на твоем Discord сервере
(может занять несколько минут, иногда час)


ШАГ 5: ЗАПУСТИТЬ БОТА!
──────────────────────────
Выполни одно из:

ВАРИАНТ А (проще):
  Двойной клик на run.bat
  (установит зависимости и запустит бота)

ВАРИАНТ Б (в PowerShell):
  npm start


ШАГ 6: Проверка успешного запуска
───────────────────────────────────
Если все правильно, ты должен увидеть:

  ✅ Connected to MongoDB
  ✅ Database initialized successfully
  🤖 Bot is ready! Logged in as YourBotName#0000
  📊 Serving 1 server(s)

Если видишь эти сообщения - ВСЕ РАБОТАЕТ! 🎉

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

════════════════════════════════════════════════════════════════

❌ "npm not found" или "node not found"
   РЕШЕНИЕ:
   1. Прочитай INSTALL_NODEJS.txt
   2. Скачай и установи Node.js LTS
   3. ПЕРЕЗАГРУЗИ PowerShell
   4. Проверь: node --version

❌ "ECONNREFUSED" при запуске
   РЕШЕНИЕ:
   1. Проверь интернет
   2. Проверь .env файл
   3. Убедись что MONGODB_URI правильный
   4. Замени <password> на реальный пароль

❌ "not authorized" при подключении MongoDB
   РЕШЕНИЕ:
   1. Проверь пароль в MONGODB_URI
   2. Скопируй правильный CONNECTION STRING с MongoDB Atlas
   3. Вставь в .env

❌ Батники не работают
   РЕШЕНИЕ:
   1. Может быть проблема с Windows защитой
   2. Используй PowerShell вместо батников
   3. Команда: npm start

❌ Команды недоступны в Discord после deploy.bat
   РЕШЕНИЕ:
   1. Дождись несколько минут (обновление может быть медленным)
   2. Перезагрузи Discord (полностью закрой приложение)
   3. Проверь что у бота есть права на создание команд
