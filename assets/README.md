# Assets

Place `beb.jpg` (the HUD template background) in this directory.

The image should be a purple HUD template (~1024x580) with designated areas for:
- Avatar (left frame, hexagonal/rhombus shape)
- Styles & SP grid (top-right)
- Attribute section (middle-right)
- Organization & Rank (bottom-right)
- XP progress bar (bottom-left)
