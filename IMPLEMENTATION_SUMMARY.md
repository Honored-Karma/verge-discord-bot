## 💡 Важные детали

### ⚠️ Не забудь!

1. **MONGODB_URI в `.env`** - это критично!
   ```env
   MONGODB_URI=mongodb+srv://user:password@cluster.mongodb.net/?appName=AppName
   ```

2. **IP в MongoDB Atlas** - добавь свой IP в Network Access

3. **npm install** - установи зависимости перед запуском

## 🔄 Если что-то пошло не так

### Ошибка подключения
```
❌ Error connecting to MongoDB
```
**Решение:** Проверь MONGODB_URI в `.env` и интернет

### Ошибка авторизации
```
❌ MongoError: not authorized
```
**Решение:** Неверный пароль в MONGODB_URI

### Ошибка инициализации
```
❌ Error initializing database
```
**Решение:** Проверь что у пользователя есть доступ к базе
